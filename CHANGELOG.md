## [1.0.0] - 2026-02-28
### Added
- React 18 + TypeScript + Vite enterprise setup
- Material UI v5 theme (indigo/teal, dark/light mode)
- Redux Toolkit + React Query state management
- JWT authentication with silent refresh interceptors
- 11 pages: Dashboard, Workouts, Exercises, Nutrition, Goals, Progress, Analytics, Profile, Users
- Recharts analytics: BarChart, LineChart, PieChart
- ESLint (airbnb-typescript) + Prettier + Husky pre-commit hooks
- Commitlint (.cjs) with Conventional Commits standard
- GitHub Actions CI (audit + build + type-check)
- Dependabot weekly dependency updates
